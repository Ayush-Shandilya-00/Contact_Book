# 📱 Contact Book

A simple, interactive terminal-based contact book application built using Python. This is my very first programming project!

## ✨ Features
* **Add Contacts**: Save a person's name and phone number.
* **View Contacts**: Display a clean list of all your saved contacts.
* **Easy Navigation**: Simple numbered menu system in the terminal.

## 🚀 How to Run the Project
1. Make sure you have **Python** installed on your computer.
2. Open your terminal in VS Code.
3. Run the following command:
   ```bash
   python main.py
   ```

## 🛠️ Technologies Used
* **Python 3** (Variables, Loops, Lists, and Dictionaries)
* **VS Code** (Code Editor)
* **Git & GitHub** (Version Control)
